難以想像 Claude Code 才剛過一歲生日。光頭（CC 之父）跟吳貓（CC+Cowork 產品總監）釋出了一段「CC 週年對話」，整理幾個核心重點。

## Agent 軍團化

軟體開發已從單一指令演進為同時啟動成百上千個 Agent 協同工作。當 AI 犯錯時，不再只是叫它重寫，而是引導它將修正轉化為長期「技能」或記錄在 claude.md 中，實現永久自主運作。

## 自動模式與常規任務

隨模型升級，不需規劃步驟即可全面啟用「自動模式（Auto Mode）」，交由專門模型進行安全過濾與審查；團隊更利用 Routines 自動監聽並修復 Github Issue。

## 職能融合與行動辦公

AI 大幅降低代碼門檻，PM、設計師和財務團隊都能直接使用 CC 變更 App，使「產品與工程」職能走向融合。工程師現在甚至能用手機遠端啟動、監控 AI 代理，實現邊走邊泡咖啡、AI 在背景寫程式的全新工作型態。

## 脈絡極簡主義

團隊不再依賴繁複的提示詞工程，改採極簡哲學，僅提供最核心的系統提示與工具，給予 AI 充分自由度以尋求最佳解決方案。

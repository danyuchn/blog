好好笑，剛剛路過彩券行，想說來下個運彩好了，再想說有沒有 Claude Code skill 處理這種事，沒想到還真的有⋯⋯ 直接接在彩券行用手機 Claude Code 現場開幹。省流板：台運賠率爛透了，誰下誰賠。

算出來的結果攤給你看。我拿世界盃 Group D 的 Australia @ USA 這場的正確比分（波膽）來算。

第一步，skill 先用市場自己的盤口反推預期進球（這是它的 edge 計算法）：從大小分加單隊進球盤，去除水錢後反推，USA 預期進球 λ≈2.0、Australia λ≈0.85，兩者相加 2.86，跟大小分盤推出的總進球一致——模型校準正確。

第二步，把台灣運彩開的賠率拿來比公平賠率，列成一張表：

![Claude 算出的各比分期望值與 Kelly 表](/blog/assets/posts/claude-code-taiwan-lottery-odds/betting-edge-table.jpg)

每個比分的運彩賠率、模型真實機率、公平賠率、每 $1 期望值、Kelly 全列出來。0:2 的運彩賠率 4.55、真實機率 11.5%、公平賠率該是 8.69，每 $1 期望值 -0.48；0:1 同樣真實機率 11.5%，運彩開 4.60，期望值 -0.47；1:2、1:1、0:3 一路下來，期望值落在 -0.34 到 -0.55。共 28 個比分，每一個的期望值都是負的，Kelly 全部寫「負」。結論直接寫在表下面：這場波膽，28 個比分沒有一個值得下。

關鍵數字在下一張：

![Claude 指出這個盤的 overround 高達 191%](/blog/assets/posts/claude-code-taiwan-lottery-odds/betting-overround-conclusion.jpg)

這個正確比分盤的總水錢（overround）是 191%。正常莊家的 1×2 盤大概是 105–110%，191% 等於莊家先把將近一半抽走。skill 算出來：正期望值的比分 0 個；連最接近的期望值都落在 -33% 到 -48%（下 $100 平均賠你 33 到 48 元）；Kelly 公式對 28 個比分全給「負值 → 不要下」。

這不是它保守，是 skill 內建的數學鐵則：f* ≤ 0 就 Don't bet。波膽就是運彩裡水錢最重的盤，莊家把熱門比分（像美國小勝 0:1 / 0:2）壓到 4.55–4.60，但真實機率只有 11.5%，等於要你用 22% 的價格買 11.5% 的東西。

最後 Claude 給我三個選擇：看完另外兩場的波膽（純當娛樂、不是投資）；改看水錢低的盤，像 1×2、大小分（水錢約 105–130%，那裡才有機會找到 value）；不划算就別下。它還補一句——想知道划不划算？答案是不划算。

來源是台灣運彩官網跟波膽玩法說明，它自己抓的。站在彩券行門口，手機點一點就把這場算完了。結論很乾脆：當娛樂可以，當投資別鬧了。

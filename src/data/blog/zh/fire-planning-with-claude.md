FIRE（Financial Independence, Retire Early）簡單說就是「存夠錢就提早退休」。這個概念在國外社群很紅，但以前要算自己離退休還有多遠，不是自己拉 Excel 套公式，就是用網路上別人做好的試算表。問題是那些模板通常很死板：你想問「如果明年股市跌三成我怎樣」或「我每年多花 20 萬會延後多久退休」，就得自己重新建模，門檻很高。

## 用對話取代公式

我用 Claude Code 做了大概一個月的個人財務規劃。把資產、負債、收支全部餵進去，直接用對話問問題：「我會不會破產？什麼狀況下會？」「每年多花 20 萬，退休會延後多久？」它自己去跑模擬、生報告。我覺得數字不對就追問：「你這個假設有根據嗎？不要自己猜。」它就去查資料修正。

跟傳統 Excel 最大的差別是，你可以像跟理財顧問聊天一樣問各種 what-if 問題，不用自己懂公式。而且 Claude Code 可以接 MCP Server，直接拉即時的匯率、市場數據進來算，不用手動更新數字。

## 累積出來的工具組

幾輪對話下來我累積了一整套工具：多幣別資產計算（我的錢分散在四種貨幣）、模擬一萬種可能的市場走勢算成功率、樂觀悲觀三種情境比較、半退休方案能提前幾年，還有自動追蹤進度。

這些不是我一次設計好的，而是每次想到新問題就加進去。「如果台幣大貶怎麼辦？」加一個匯率壓力測試。「如果我提前半退休只做兼職？」加一個半退休模擬。工具是跟著問題長出來的。

## 亞洲視角

後來覺得這些方法不該只有我能用，我就把個人資料全部拔掉，做成公開的 Claude Code Skill。20 篇指南加三個參考程式，涵蓋退休目標怎麼算、錢怎麼安全慢慢領、市場大跌怎麼應對、跨國多幣別怎麼追蹤。

其中有一塊是我覺得特別重要的亞洲視角：健保怎麼銜接、家裡長輩的財務責任、簽證居留問題。這些國外的 FIRE 工具幾乎不會提，但對在亞洲生活的人來說是最現實的考量。

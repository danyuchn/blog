法務查專利侵權，原本要好幾天。

上完 Claude Code 課之後，一天做完，而且真的抓到了，已經送去走法律程序了。

C 把這件事傳訊息告訴我的時候，我感到不可思議。

## 從學員變成客戶

C 原本是我 AI 工作坊的學員，後來變成我企業培訓的客戶。她是一家幾十人公司的法務。

課後她找上我們，說公司有需求，我們上線視訊會議，她把法務的故事說給我聽：

查專利侵權這件事，以前是法務要自己手動查，好幾天才跑完一輪。套上 Claude Code 之後，一天就做好。但讓我覺得驚訝的不是這個數字。而是她說，真的有抓到實際侵權案例，已經送去走法律程序了。

## 為什麼這個案例讓我在意

我一直認為，Claude Code 不是只有工程師能用的工具。這話以前我說出來，很多人都打個問號，只有我自己相信，因為我就是那一個非工程師的使用者。

C 的例子，讓那個問號消失了。

當法務可以用 Claude Code 查專利侵權、當一個幾十人公司的法務部門開始把它當成日常工具——這就不再是「個別案例」，而是一個工作流的範式轉移。

## 不是「AI 取代法務」

要強調一件事：Claude Code 沒有取代 C，是 C 用 Claude Code 把原本要花 3-5 天的工作壓到 1 天。剩下的 2-4 天，她可以多看幾個案子，或者把案件研究得更深。

這跟「AI 取代人類」的劇本完全不一樣。實際上是 C 變成「會用 AI 的法務」，而那些不會用的法務開始落後。

工具會變，但有人用工具、有人沒用，這個分野永遠存在。

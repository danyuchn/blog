有一本書可以推薦，叫做 [Street Smarts](http://www.books.com.tw/products/0010705054)（街頭智慧），作者 Norm Brodsky。

## 做成 Skill 之後的改變

我把這本書的核心框架做成了 Claude Code 的 Skill。每次有新的合作機會進來，我就讓這個 Skill 幫我跑一遍分析。

結果發現，很多看似很好的合作機會，其實毛利算下去都是賠錢的。對方吹出的泡泡都是假的。

這本書教的不是什麼高深理論，就是幾個很基本的問題：

- 這筆生意的毛利是多少？
- 扣掉隱性成本後，有效時薪是多少？
- 對方的承諾有沒有實質擔保？
- 規模化之後成本結構怎麼變？

## 心中有底，波瀾不驚

有了這本書的理性分析框架後，我接合作案變得很冷靜。

因為算出來的數字就在那邊，底線就在那邊。心中有底，波瀾不驚。對方嘴上再怎麼跑火車，那是他的事。自己的 scope 踩好就好。

最近一個實際案例：有位學員想要 1 對 1 諮詢，預算只有報價的六折。算下去，備課加上後續跟進的隱性時間成本，有效時薪只剩三分之一。不划算就是不划算，沒什麼好糾結的，直接導向課程就好。

## 為什麼要做成 Skill

把書做成 Skill 的好處是：每次決策不用重新回想書裡寫了什麼。Skill 會直接問對的問題、套對的公式、給出數字。人容易在對方的話術和關係壓力下動搖，但數字不會。

這本書特別適合自由工作者和一人公司。不是教你怎麼賺大錢，是教你怎麼不賠錢。

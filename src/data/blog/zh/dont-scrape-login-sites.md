其實需要登入的網站，我一律建議一般使用者不要跟它硬碰硬，直接找第三方爬蟲資料庫 API。

## 你的 token 就是你的帳號

原因很簡單，你的 token 就是你的帳號。玩得過火，admin 那邊「二向箔」一丟，你的帳號就沒了。

二向箔是《三體》的梗，意思是降維打擊——對方不跟你講道理，直接把你封掉。

我就有一個 2012 年到現在的珍貴老帳號，被我自己手賤搞沒。

## 交給專業的爬蟲 API

你說用小號？產小號的時間成本可以拿去做很多事。

所以交給專業爬蟲 API，付一點小錢轉嫁風險吧。小號爬蟲他們專業的，反爬對幹他們沒在怕的。

技術上你當然可以叫 agent 登入後台一頁頁爬，但能做到不等於該自己做。

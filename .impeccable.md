# Impeccable Design Context

## Design Context

### Users
讀者是對 AI 工具有興趣的中文使用者（以台灣、香港為主），在電腦與手機上閱讀的比重相當。核心受眾是**學員與潛在客戶**——他們想看的不是教學文件，而是「一個真實的人怎麼用 AI 做東西、踩了什麼坑、有什麼觀點」，同時在評估這個人值不值得信任。文章連結經常被貼到 Threads / Facebook，**社群分享卡（OG 圖）是第一印象的主戰場**，與站內同等重要。

**Author persona**: Dustin，曼谷的 GMAT 教師轉 AI 產品建造者。非工程師背景，用 Claude Code 和 Gemini 把想法實現。寫的是真實經歷，不是技術文件。

### Brand Personality
**三個詞**：個人・真實・有溫度

- 像在跟朋友講話，不是在做簡報
- 有觀點，敢說不喜歡什麼
- 不假裝自己是工程師，但認真把事情做好
- 文字密度高，但不枯燥

### Aesthetic Direction
**核心方向**：雜誌墨黑 × 瑞士網格——印刷雜誌的權威感配上國際主義排版的秩序，個人聲音由文字與襯線字型承載（2026-07 定案，取代先前的 warm editorial）

**正面參考**：
- 獨立印刷雜誌的版面：黑白高對比、粗規線、襯線大標
- 瑞士國際主義排版：嚴格網格、大留白、單一高飽和強調色、大寫字距標籤
- 報紙頭版的資訊密度：日期欄、eyebrow、規線分區

**明確反對（Anti-references）**：
- Medium：白底黑字超寬，無聊，沒個性
- AI cyber/neon：深色 + 紫色漸層 + 發光邊框
- 卡片堆疊 portfolio 感：圓角陰影卡片，像商業模板
- AI 生成預設風：warm cream 底 + serif 標題 + terracotta 強調色（前一版的样子，已畢業）
- 冷淡到失去人味：黑白骨架之上，襯線內文與第一人稱文字負責保溫

**主題**：Light 和 Dark 都要精緻。Dark mode 是「墨黑印刷品」——近黑紙面、米白墨色、橘紅強調——不是「駭客終端機」

### Design Tokens（實作基準）
- **色彩**：light `#ffffff` / `#111111`、accent `#c8371e`（朱紅）；dark `#141414` / `#f2efea`、accent `#ff6a4d`。全站唯一強調色，語意色（diff 綠紅）除外
- **字型**：標題 Noto Serif TC（700/900，webfont，config＋`<Font>` 兩步掛載）；內文系統黑體 stack（蘋方/思源黑體）；code 用 Google Sans Code
- **規線語彙**：粗規線（2px+ foreground）做結構分區（header、hero、footer），細規線（border token）做列表分隔；一律實線，不用虛線
- **卡片**：瑞士網格式——日期獨立左欄、標題襯線粗體、所有卡片顯示 description；不依賴 `last:`/`nth-child`（LangSwitcher 會 display:none 隱藏另一語言的卡）
- **標籤**：方角邊框膠囊、大寫、字距加寬；圓角一律 0（含 Pagefind `--pagefind-ui-border-radius`）

### Design Principles
1. **文字第一**：排版決定一切。字型、尺寸、行距比視覺裝飾更重要
2. **對比就是個性**：黑白高對比、大標大留白，靠版面張力而非裝飾記住這個站
3. **留白是設計**：不塞滿，讓內容呼吸，閱讀體驗是核心
4. **顏色要克制**：朱紅是唯一強調色，出現得少才有力；內文級用色須過 AA 對比
5. **兩者皆精緻**：Light mode 是白報紙，Dark mode 是墨黑印刷品，不是事後補丁
6. **站內站外一致**：OG 分享卡與站內同一套視覺語言（黑白＋朱紅＋襯線大標），改版必同步兩邊
